# Умная диагностика

Сервис быстрого анализа табличных данных (CSV / XLSX). Загружаете файл — получаете саммари, дашборд и PDF-отчёт с выводами и рекомендациями.
[Скриншот карточки загрузки данных]()
## Пользователь

**Методисты в образовании**, педагоги, аналитики — все, кому нужно быстро оценить качество данных, увидеть статистику и получить готовые выводы без настройки сложных аналитических систем.

Проект создан для профориентационной работы со школьниками: анализа результатов диагностик, выявления проблемных зон и отслеживания динамики обучения.

## Основная функция

**Загрузка табличного файла → автоматический анализ → пакет отчётов**

Система сама определяет типы столбцов, считает метрики, находит проблемы и формирует три вида отчётов. Конвейер обработки:

```
[CSV/XLSX] → [1. Ingest] → [2. Analyze] → [3. Report]
                  ↓               ↓               ↓
           Валидация        Статистика        Markdown
           Очистка          Проблемы          Дашборд
           Конвертация      Оценки            PDF
```

### Детальные возможности

- **Загрузка данных** — CSV, XLSX, XLS (до 50 МБ) через веб-интерфейс
- **Автоопределение типов столбцов** — числовые, категориальные, даты, текст, пустые
- **Очистка данных** — автоопределение кодировки (UTF-8, CP1251, CP866, Latin1), определение разделителя (`,` `;` `\t`), нормализация пропусков, замена бесконечностей
- **Статистический анализ** — количество строк/столбцов, среднее, медиана, стандартное отклонение, min/max, Q1/Q3, гистограммы
- **Поиск проблем** — пропуски (с оценкой серьёзности), выбросы (IQR), дубликаты строк, аномалии по датам
- **Анализ оценок (A/B/C)** — для образовательных данных: автоматическое обнаружение столбцов с оценками (включая русские буквы А/В/С), расчёт:
  - Распределение оценок по каждому критерию
  - **Уровень обученности** (% оценок A + B)
  - **Уровень самостоятельности** (% оценок A)
- **Сравнение периодов** — при загрузке второго файла (например, Q4 vs Q1) рассчитывается дельта и процент изменения метрик
- **Автоматические выводы** — формируются на основе метрик качества данных и обнаруженных проблем
- **Приоритезированные рекомендации** — действия по улучшению с ожидаемым эффектом, сортированные по приоритету (High / Medium / Low)
- **Веб-интерфейс** — загрузка файлов через браузер (localhost:5000), просмотр отчётов
- **CLI-режим** — запуск конвейера из командной строки с опциями `--compare`, `--date-col`, `--value-cols`

## Форма результата

Для каждого загруженного файла генерируется три отчёта (сохраняются в `reports/<имя_файла>/`):

| Формат | Файл | Содержание |
|---|---|---|
| **Markdown-саммари** | `YYYY-MM-DD_<name>_summary.md` | Таблицы со статистикой, проблемами, выводами и рекомендациями; при образовательных данных — анализ оценок |
| **HTML-дашборд** | `YYYY-MM-DD_<name>_dashboard.html` | Самодостаточная веб-страница: KPI-карточки, графики Chart.js (распределение, пропуски, сравнение оценок), таблица проблем, список рекомендаций |
| **PDF-отчёт** | `YYYY-MM-DD_<name>_report.pdf` | Версия для печати с теми же секциями, цветовой индикацией серьёзности, кириллическими шрифтами (DejaVu) |

Дашборд автоматически подстраивает отображение под тип данных: для общих данных — гистограмма + график пропусков; для образовательных оценок — сравнительная диаграмма оценок + горизонтальный график самостоятельности.

## Технологии

| Компонент | Технология |
|---|---|
| **Язык** | Python 3 (совместим с 3.9 и 3.12) |
| **Веб-фреймворк** | Flask (Werkzeug) |
| **Анализ данных** | Pandas, NumPy |
| **Excel → CSV** | OpenPyXL |
| **Дашборд (графики)** | Chart.js 4.x + chartjs-plugin-datalabels (CDN) |
| **PDF** | FPDF + DejaVu fonts |
| **Фронтенд** | HTML5 + CSS3 + vanilla JS (единый файл) |
| **Управление зависимостями** | pip |
| **CLI / Git** | GitHub CLI, Git |

## Особенности

1. **Образовательная направленность** — специализированная логика для учебных данных: автоматическое распознавание оценок A/B/C (включая кириллицу), педагогические метрики
2. **Двухрежимный дашборд** — переключает отображение в зависимости от типа данных без изменения кода
3. **Сравнение двух файлов** — дельта и динамика между периодами
4. **Устойчивость к кодировкам** — автоматический перебор UTF-8 → CP1251 → CP866 → Latin1 + автоопределение разделителя
5. **Лёгкий запуск** — без БД, Docker и внешних сервисов, всё на localhost
6. **Полностью на русском** — отчёты, интерфейс, выводы

## Быстрый старт

```bash
# Установка зависимостей
pip install flask pandas numpy openpyxl fpdf2

# Запуск веб-сервера
python start_server.py
# → http://127.0.0.1:5000/

# Или запуск конвейера напрямую
python scripts/run_pipeline.py data/raw/example.csv
```
