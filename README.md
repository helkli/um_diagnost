# Умная диагностика

Сервис быстрого анализа табличных данных (CSV / XLSX). Загружаете файл — получаете саммари, дашборд и PDF-отчёт с выводами и рекомендациями.

## Пользователь

Методисты в образовании, педагоги, аналитики — все, кому нужно быстро оценить качество данных, увидеть статистику и получить готовые выводы без настройки сложных аналитических систем.

## Основная функция

**Загрузка табличного файла → автоматический анализ → пакет отчётов**

Система сама определяет типы столбцов, считает метрики, находит проблемы и формирует три вида отчётов.

## Форма результата

| Формат | Описание |
|---|---|
| **Markdown-саммари** | Краткий текстовый отчёт: статистика, проблемы, выводы, рекомендации |
| **HTML-дашборд** | Интерактивная веб-страница с KPI, графиками, таблицей проблем |
| **PDF-отчёт** | Печатная версия дашборда |

## Функционал

- **Загрузка данных** — CSV, XLSX, XLS (до 50 МБ)
- **Автоопределение типов** — числовые, категориальные, даты
- **Очистка данных** — обработка пропусков, нормализация
- **Статистический анализ** — среднее, медиана, квартили, распределения
- **Поиск проблем** — пропуски, выбросы (IQR), дубликаты, аномалии
- **Анализ оценок** — для образовательных данных (A/B/C), расчёт уровня обученности и самостоятельности
- **Сравнение периодов** — дельта и динамика показателей при загрузке второго файла
- **Рекомендации** — приоритезированные действия по улучшению качества данных
- **Веб-интерфейс** — загрузка файлов через браузер (localhost:5000)

## Технология

| Компонент | Технология |
|---|---|
| Язык | Python 3 |
| Веб-фреймворк | Flask |
| Анализ данных | Pandas, NumPy |
| Excel → CSV | OpenPyXL |
| Дашборд | HTML + CSS + JS (Chart.js) |
| PDF | браузерная печать HTML |
| CLI | GitHub CLI, Git |

## Структура проекта

```
um_diagnost/
├── data/
│   └── raw/              # Исходные CSV-файлы
├── scripts/
│   ├── ingest/           # Загрузка, валидация, очистка CSV
│   ├── analyze/          # Статистика, аномалии, тренды, оценки
│   └── report/           # Генерация саммари, дашборда, PDF
│   └── run_pipeline.py   # Оркестратор конвейера
├── reports/              # Сгенерированные отчёты (по папкам на каждый файл)
├── web/
│   ├── app.py            # Веб-сервер Flask
│   └── index.html        # Интерфейс загрузки
├── docs/
│   ├── specs/            # ТЗ и описания форматов
│   └── examples/         # Примеры отчётов
├── start_server.py       # Скрипт запуска сервера
└── .gitignore
```

## Конвейер обработки

```
[CSV/XLSX] → [1. Ingest] → [2. Analyze] → [3. Report]
                  ↓               ↓               ↓
           Валидация        Статистика        Markdown
           Очистка          Проблемы          Дашборд
           Конвертация      Оценки            PDF
```

## Быстрый старт

```bash
# Установка зависимостей
pip install flask pandas numpy openpyxl

# Запуск веб-сервера
python start_server.py
# → http://127.0.0.1:5000/

# Или запуск конвейера напрямую
python scripts/run_pipeline.py data/raw/example.csv
```
