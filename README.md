# Умная диагностика

Сервис быстрого анализа табличных данных (CSV). На входе — файл с данными, на выходе — саммари, изменения, проблемы, выводы и дашборд.

## Структура проекта

```
um_diagnost/
├── data/
│   ├── raw/              # Исходные CSV-файлы без изменений
│   └── processed/        # Очищенные и подготовленные данные
├── scripts/
│   ├── ingest/           # Загрузка, валидация, парсинг CSV
│   ├── analyze/          # Расчёты: статистика, изменения, аномалии, тренды
│   └── report/           # Генерация отчётов, графиков, дашбордов
├── reports/
│   ├── summaries/        # Краткие текстовые саммари по analysed данные
│   ├── dashboards/       # Визуальные дашборды (HTML, PNG, PDF)
│   └── recommendations/  # Выводы и рекомендации на основе анализа
├── docs/
│   ├── specs/            # ТЗ, описания форматов, требования
│   └── examples/         # Примеры входных данных и эталонных отчётов
├── tmp/                  # Временные файлы (в gitignore)
├── claude.md             # Настройки ассистента
└── README.md
```

## Конвейер обработки

1. **ingest** — загрузка CSV, проверка схемы, очистка
2. **analyze** — расчёт метрик, поиск изменений/аномалий
3. **report** — формирование саммари, дашборда, рекомендаций
